# Changelog

All notable changes to this project will be documented in this file.

- CI pipeline with GitHub Actions (build, lint, type-check)
- Dependabot config for automatic dependency updates
- CodeQL security scanning
- Release workflow with npm publish
- Issue templates (bug report, feature request)

- ESLint + Prettier configuration
- `.npmignore` for clean npm publishes
- Contributing guidelines

## [1.0.0] - 2025-01-01

### Added
- Initial MCP server with 10 memory management tools
- Obsidian vault integration with markdown-based storage
- Session management (start/end with summaries)
- Full-text memory search across all entries
- Project tracking with progress and context updates
- Auto-generated wiki-links for Obsidian Graph View
- Global index with cross-project navigation
